# KidTrack

**Version 2.1.0** | Build 4 | January 2026

KidTrack is an Android application designed to help parents and guardians track their children's daily activities and set reminders. The app features a user-friendly interface and provides various functionalities to manage children's activities effectively.

## Features

- **User Profiles**: Create and manage profiles for each child, allowing personalized tracking of activities and reminders.
- **Activity Tracking**: Add, view, and manage daily activities for children, including descriptions and notes.
- **Predefined Tasks**: Choose from 26 common activities (School, Health, Routine, Social, Activity, Chore).
- **Reminders and Notifications**: Set reminders for activities with sound alerts and snooze functionality.
- **Day-Before Alerts**: Get notified 1 day before important events.
- **Dashboard**: A central dashboard that displays upcoming activities and overdue tasks at a glance.
- **Reports**: Generate weekly summaries of completed vs. missed activities to monitor progress.

## Technologies Used

- Kotlin: The primary programming language for Android development.
- Android Jetpack: Utilizes components like ViewModel and LiveData for efficient data management.
- Room: A persistence library for managing local database operations.
- Material Design: Ensures a kid-friendly and visually appealing UI/UX.
